KoBar is a versatile platform designed to host a rich ecosystem of plugins, seamlessly integrating powerful tools directly into your daily workflow. As a core extension for this ecosystem, **KoCalendar** transforms KoBar into a powerful, interactive scheduling hub.

## What is KoCalendar?

KoCalendar is an interactive, draggable calendar popup natively integrated into the KoBar environment. It empowers users to quickly view, manage, and schedule local events while synchronizing with KoBar's internal ToDo system to seamlessly track deadlines right from their desktop sidebar.

## Key Features

- **Smart Screen Positioning**: Intelligently auto-adjusts its location based on your KoBar orientation (vertical/horizontal) and edge position, preventing it from clipping out of screen boundaries.
- **Interactive Drag Mechanism**: Effortlessly move the calendar anywhere on your screen using its drag handle.
- **Flicker-Free Grid**: A beautifully designed 42-day calendar matrix with integrated glassmorphism effects tailored for both Windows and macOS.
- **Fast Navigation Overlay**: Features a custom Month & Year Picker for rapidly jumping across timelines.
- **Inline Event Management (CRUD)**: Easily add, edit, or delete events, select unique color tags, and add meeting URLs.
- **Desktop Notifications**: Seamless integration with the host OS for timely alerts.
- **JSON Bulk Import**: Instantly import holidays or other bulk events from `.json` files.
- **Meeting Quick Links**: Directly join video calls from the built-in Agenda view via KoBar's IPC bridge.

## Configuration & Usage

- **Opening the Calendar**: Click the KoCalendar icon from the KoBar sidebar to launch the pop-up.
- **Navigation**: Use the quick action buttons to navigate between months, or click the header to use the fast Month/Year Picker.
- **Creating Events**: Double-click any day to instantly create a new event, or use the "+" icon in the Agenda view.
- **Agenda View**: Click on any date to open the Agenda view at the bottom and see all scheduled items and ToDo deadlines for that day.
- **Importing Holidays**: Use the download icon on the top right to upload a JSON file containing bulk events or holidays, and easily map them to a specific color tag.
